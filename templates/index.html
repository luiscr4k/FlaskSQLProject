Flask como framework

Empleamos flask como framework WSGI debido a su practicidad y minimalismo a la hora de comprender toda su libreria de funciones, este Framework de Python permite crear aplicaciones web rápidamente y con un mínimo número de líneas de código, adicionalmente, cuenta con muchas librerías basadas en la especificación WSGI de Werkzeug y el motor de templates Jinja2.
